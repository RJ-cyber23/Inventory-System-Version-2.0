# Instructions

In order to work with the files in this repository, you need to download the **draw.io** platform to read and write the `.drawio` file.

### Recommended Platforms:
* **Web:** [app.diagrams.net](https://app.diagrams.net/)
* **Desktop:** [diagrams.net/download](https://www.diagrams.net/download)
* **VS Code:** Install the "Draw.io Integration" extension.
